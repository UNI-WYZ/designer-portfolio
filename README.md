# Designer Portfolio

Personal design portfolio for UNI, focused on original IP design, brand visual systems, commercial landing materials, posters, and creator identity.

## Project Type

This is a static website. The entry file is `index.html`, with local styles and scripts:

- `index.html`
- `styles.css`
- `script.js`
- `assets/`

## Deploy

### Vercel

Import this repository into Vercel and use the default static project settings.

- Build command: leave empty
- Output directory: `.`

### Netlify

Import this repository into Netlify.

- Build command: leave empty
- Publish directory: `.`

## Update Workflow

After editing the site locally:

```bash
git add .
git commit -m "Update portfolio"
git push origin main
```

Vercel or Netlify will automatically redeploy after each push to `main`.
